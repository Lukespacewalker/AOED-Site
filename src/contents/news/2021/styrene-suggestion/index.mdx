# สำหรับประชาชน
1. ในระยะนี้ไม่ควรวิ่งออกกำลังกายจนกว่าสภาพอากาศจะดีขึ้น
2. ถ้ามีอาการแสบตาจมูกปากหรือคอให้เข้าไปในบ้านปิดประตูหน้าต่างเปิดเครื่องปรับอากาศหรือถ้าไม่มีเครื่องปรับอากาศให้หลบอยู่ในบ้าน
3. ถ้ามีอาการหอบไอเหนื่อยให้ไปโรงพยาบาลหรือย้ายไปยังสถานที่ซึ่งปราศจากหมอกควัน
4. ให้สวมหน้ากากอนามัยเวลาออกนอกอาคารบ้านเรือนตลอดเวลา
5. ในผู้ที่มีอาการหอบหืดโรคถุงลมโป่งพองโรคกล้ามเนื้อหัวใจขาดเลือดหรือโรคควโรคถุงลมโป่งพองโรคกล้ามเนื้อหัวใจขาดเลือดหรือโรคความดันโลหิตสูงมากๆถ้ามีอาการหายใจลำบากไอมากแสบคอแสบตาหรือแสบจมูกให้ามดันโลหิตสูงมากๆถ้ามีอาการหายใจลำบากไอมากแสบคอแสบตาหรือแสบจมูกให้รีบหลบเข้าบ้านเรือนหรือมาที่โรงพยาบาล
6. อาการของกลุ่มอาการพิษสไตรีนคือปวดมึนศีรษะคลื่นไส้อาเจียนเดินเซเกิดจากการได้รับสารพิษเป็นจำนวนมาก
7. ให้ติดตามข่าวสารเรื่องระดับมลพิษอย่างใกล้ชิด
8. ถ้าท่านเป็นคนกลุ่มที่เสี่ยงคือเป็นเด็กเล็กผู้สูงอายุมีโรคประจำตัวหรือกินยาบางอย่างให้รีบพบแพทย์ถ้ามีอาการผิดปกติ
9. ผู้ที่สงสัยว่าจะมีอาการที่เกิดควันพิษให้ติดต่อรักษาตัวกับแพทย์

ถ้ามีอาการดังกล่าวและกลับไปอยู่ภูมิลำเนาให้บอกแพทย์ที่ตรวจด้วยว่ามีการสัมผัสสารพิษด้วย สำหรับพิษเรื้อรังถ้ามีอาการผิดปกติและสงสัยว่าอาจจะเกิดจากพิษสไตรีนให้ปรึกษาแพทย์

# สำหรับผู้เผชิญเหตุ
1. อันตรายที่เกิดขึ้นนั้นจะมากหรือน้อยขึ้นกับการสัมผัสโดยขึ้นกับความรุนแรงของสารเคมีขนาดของสารเคมีคือใกล้แหล่งสารเคมีมากขนาดได้และระยะเวลาการสัมผัสคือระยะเวลาที่ทำงานในบริเวณนั้นรวมถึงการป้องกันตนเองโดยเครื่องป้องกันอันตรายได้แก่หน้ากากป้องกันทางเดินหายใจเป็นต้นดังนั้นผู้ที่ทำงานใกล้จุดเกิดเหตุจะได้รับความผิดขนาดมากโดยเฉพาะเมื่อไม่ใส่เครื่องป้องกันตนเองที่เหมาะสมและทำงานเป็นระยะเวลานานดังนั้นควรใส่เสื้อผ้ามิดชิดใส่หน้ากากที่มีไส้กรองสารเคมีหรืออย่างน้อยที่สุดชนิด N95
2. ไม่ควรเป็นผู้ที่เป็นโรคหืดโรคปอดชนิดอื่นเช่นโรคถุงลมโป่งพองโรคหลอดลมอักเสบหรือวัณโรคปอดโรคหัวใจโดยเฉพาะโรคกล้ามเนื้อหัวใจขาดเลือด
3.ถ้ามีอาการหอบแน่นหน้าอกไอเจ็บหน้าอกอ่อนเพลียมึนงงศีรษะมากคันตามตัวมากแสบตาแสบคอมากควรปรึกษาแพทย์
4. อาการของกลุ่มอาการผิดสไตรีนคือปวดมึนศีรษะคลื่นไส้อาเจียนเดินเซเกิดจากการได้รับสารพิษเป็นจำนวนมาก
5. ไม่ควรรับประทานอาหารหรือดื่มน้ำในบริเวณปฏิบัติงานหลังเข้าทำงานในพื้นที่ไฟไหม้ควรเปลี่ยนเสื้อผ้าอาบน้ำสระผมทำความสะอาดตัวเองให้เรียบร้อยก่อนพบปะกับสมาชิกครอบครัวหมอทันมั้ยค่ะ
6.ทั้งนี้เมื่อเสร็จสิ้นภารกิจควรมาพบแพทย์เพื่อทำการตรวจร่างกายตามความเสี่ยงที่พบและติดตามตรวจร่างกายเป็นระยะตามที่แพทย์กำหนด

ถ้ามีอาการดังกล่าวและกลับไปอยู่ภูมิลำเนาให้บอกแพทย์ที่ตรวจด้วยว่ามีการสัมผัสสารพิษด้วย สำหรับพิษเรื้อรังถ้ามีอาการผิดปกติและสงสัยว่าอาจจะเกิดจากพิษสไตรีนให้ปรึกษาแพทย์
